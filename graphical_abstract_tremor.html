Frekvens
Klinisk bild / utlösande
Nyckeldiagnostik
Essentiell tremor (ET)
4–12 Hz
Bilateral handtremor vid rörelse/postur. Lindras av alkohol. Ofta hereditet.
Kinetisk ≥ postural. Uteslut PS, dystoni, metabol orsak.
Parkinson-tremor
4–6 Hz
Asymmetrisk vilotremor. "Pill-rolling". Minskar vid rörelse.
Levodopa-test. DaTscan vid osäkerhet.
Dyston tremor
Variabel
Associerat med dystonisk felställning. Ofta nack/arm. "Nullpunkt".
Undersök noggrant för dystoni i annan kroppsdel.
Cerebellär tremor
<5 Hz
Intention-tremor, värre nära målet. Ataxisymtom, dysmetri.
MR hjärna/ryggmärg. Tänk MS, SCA, alkohol.
Ortostatisk tremor
13–18 Hz
Ben-tremor vid stående, försvinner vid gång. Otrygghetskänsla.
EMG (16 Hz-mönster). Kliniskt ofta förbisett.
Förstärkt fysiologisk
8–12 Hz
Postural. Utlöst av: koffein, stress, tyreoidos, läkemedel (valproat, litium, β-agonister), etanolabstinens.
Anamnes, lab (TSH, B12, Mg). Läkemedelslista!
Holmes tremor
<4,5 Hz
Svår blandad tremor (vila+postur+intention). Strukturell hjärnskada.
MR hjärna. Cerebellum/thalamus-lesion vanligast.
Farmakologisk behandling
Essentiell tremor
  • Propranolol 40–120 mg/d (förstaval)
  • Primidon 62,5 mg till natten (titrera)
  • Kombination vid otillräcklig effekt
  • Alternativ: gabapentin, topiramat, nadolol
Parkinson-tremor
  • Levodopa (rigiditet/brady, tremor variabel respons)
  • Amantadin – måttlig tremor-effekt
  • Klozapin 12,5–75 mg – vid terapiresistent tremor
  • DBS STN/GPi vid invalidiserande tremor
Dyston tremor
  • Botulinumtoxin (förstaval för nack/fokal dystoni)
  • Klonazepam, trihexyfenidyl
  • DBS GPi vid generaliserad dystoni + tremor
Cerebellär tremor
  • Primärt: behandla grundsjukdom (MS etc.)
  • Klonazepam, gabapentin (begränsad effekt)
  • DBS VIM – kan prövas i enstaka fall
Förstärkt fysiologisk tremor
  • Elimina utlösande faktor (läkemedel, metabol)
  • Propranolol vid kvarständande symtom
Ortostatisk tremor
  • Klonazepam (förstaval)
  • Gabapentin, primär 10 mg/d levodopa-prov
  • Ofta svårbehandlad

MRgFUS – Fokuserad ultraljud (talamotomi)
  • Icke-invasiv lesionsmetod (VIM)
  • Indikation: ET/PS-tremor, unilateralt, terapiresistent
  • Kräver adekvat skalltransmission (DI-index)
  • Irreversibel – noggrann patientselektering
  • Bilateralt: diskuteras (risk dysartri/ataxi)
